# Python With ChatGPT Workspace

This workspace currently contains a React + TypeScript login demo app built with Vite.

## Project Structure

- `react-login-app/`: Main frontend project.

## Quick Start

1. Open a terminal in `react-login-app`.
2. Install dependencies:

```bash
npm install
```

3. Start the dev server:

```bash
npm run dev
```

4. Build for production:

```bash
npm run build
```

5. Lint the project:

```bash
npm run lint
```

## README Maintenance Policy

When any project behavior changes, update documentation in the same change set.

Minimum rule:

- Update `react-login-app/README.md` whenever features, scripts, credentials, UX flow, setup, or dependencies change.
- Keep this root README updated if folders are added, renamed, or removed.

Use this checklist for every change:

1. What changed?
2. Why it changed?
3. How to run/test it now?
4. Any new constraints, credentials, or caveats?
5. Which README section was updated?

If there are no user-facing or workflow changes, add a short note in the PR/commit message that README review was completed and no update was required.
