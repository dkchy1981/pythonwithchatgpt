# Python With ChatGPT Workspace

This workspace currently contains a React + TypeScript login demo app built with Vite.

## Project Structure

- `react-login-app/`: Main frontend project.

## Quick Start

1. Open a terminal in `react-login-app`.
2. Install dependencies:

```bash
npm install
```

3. Start the dev server:

```bash
npm run dev
```

4. Build for production:

```bash
npm run build
```

5. Lint the project:

```bash
npm run lint
```

## README Maintenance Policy

For this workspace, README updates are strict: when any code changes, update relevant README files in the same change set.

Minimum rule:

- Update all relevant README files whenever code changes.
- Keep `react-login-app/README.md` current for app behavior, scripts, setup, and dependencies.
- Keep this root README updated if folders are added, renamed, removed, or repurposed.

Use this checklist for every change:

1. What changed?
2. Why it changed?
3. How to run/test it now?
4. Any new constraints, credentials, or caveats?
5. Which README section was updated?

If no content changed in a README after review, add a short note in the PR/commit message that strict README review was completed.
