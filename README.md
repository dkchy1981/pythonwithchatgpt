## Installation
1. Clone the repository:
   ```bash
   git clone https://github.com/dkchy1981/pythonwithchatgpt.git
   cd pythonwithchatgpt
   ```
2. (Optional) Create a virtual environment:
   ```bash
   python -m venv .venv
   source .venv/bin/activate  # On Windows: .venv\Scripts\activate
   ```
3. Install dependencies when they are added:
   ```bash
   pip install -r requirements.txt
   ```
